**备忘**
- 本地预览命令：docsify serve docs
- `!>` ：一段重要的内容
- `?>` ：普通提示
- 增加封面：在 `index.html` 中添加下方代码，然后在根目录下创建 _coverpage.md 文件
```
<script>
    window.$docsify = {
      coverpage: true
    }
</script>
```
- `.nojekyll` 用于阻止 GitHub 仓库忽略掉下划线开头的文件
- 用 Obsidian 编辑的话要在文件夹下新建 `.gitignore` 并写入 `.obsidian`
